# Skincare-Assistant-App
The Skincare Assistant App is an AI-powered mobile application designed to analyze skin conditions and provide personalized skincare recommendations. It helps users identify skin issues such as acne, dryness, pigmentation, wrinkles, and sensitivity using advanced technologies like computer vision and machine learning.

# Abstract
This project presents an AI-powered Skincare Assistance App designed to help users identify skin types, track skin health, and generate personalized skincare routines. The system uses image analysis and expert-backed algorithms to provide recommendations and monitor progress. Implementation results show effective skin type classification and routine suggestions, enhancing skincare adherence and awareness. This project demonstrates the integration of AI, computer vision, and user-friendly interfaces for practical personal care solutions.

# Introduction
Skin health is vital for overall well-being, but most users struggle to select appropriate skincare products. The app aims to analyze skin condition, suggest routines, and track improvement. It addresses the growing demand for personalized skincare, leveraging technology to simplify self-care and prevent adverse reactions from unsuitable products.

# Literature Review
Existing solutions include apps for acne tracking, product recommendations, and AI skin analysis. Most lack holistic routine building or fail to combine image analysis with ingredient safety checks. This project integrates AI, dermatology insights, and progress tracking for a more comprehensive solution.

# Methodology
The system works by taking a user selfie, applying AI-based skin analysis to detect skin type and conditions, and generating a personalized skincare routine using ingredient safety and dermatology guidelines. Users can track daily skin progress, receive reminders, and view routine effectiveness over time.
